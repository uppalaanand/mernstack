### Steps to create Backend
    1. Generate package.json
        npm init -y                 -npm -> node package manager

    2. Create HTTP Server
        a. Install & import express module
            npm install                     - used to install any modele
            npm install express               - it will install express module & stores in node_modules
        
        b. Import express module

    POST  --- Create a new resource
